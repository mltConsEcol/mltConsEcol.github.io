Research Tools I Use
R Statisics Package
A freeware statistics program with a wide variety of analyses built in, and countless others for which downloadable packages are available and easy to use.
SAGA GIS
A freeware GIS package that has a wide variety of capabilities, and can take advantage of 64-bit, multi-core processors. The built-in modules make easy work of processing remotely sensed raster data such as satellite imagery and digital elevation models. It can also process LIDAR point cloud data, and works well to manipulate vector and table data formats in various ways.
Notepad++
A free text editor for windows that provides a great deal of utility and flexibility for many different languages
NppToR: R in Notepad++
An awesome add-in that enables you to run R Code directly from Notepad++
Manifold GIS
An independent GIS package that is relatively cheap, fast, incredibly flexible, and has an amazing user support forum
